# Rust-Contactbook

CLI-Tool to save, edit and search for contacts in mongodb database.
MongoDB and MongoExpress running as containers with docker compose.

## Install

1. Clone repository
2. Cd in directory
3. Spin up docker compose

```bash
docker compose up
```

4. Run Cargo project

```bash
cargo run
```

5. Enjoy
